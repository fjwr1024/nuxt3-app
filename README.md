# Nuxt3(Stable 版) example

Nuxt.js の example
公式 [Nuxt 3 documentation](https://nuxt.com/docs/getting-started/introduction)

## 環境構築

### 1. `git clone`

```bash
git@github.com:fjwr1024/nuxt3-app.git
```

### 2. `yarn(npm) install`

```bash
# yarn
yarn install

# npm
npm install
```

### 3. `yarn(npm run) dev`

`yarn(npm run) dev`でローカルサーバーを立ち上げ。

```bash
yarn dev
```

```bash
npm run  dev
```

## 開発サーバー

http://localhost:3000

```bash
npm run dev
```

## Production

```bash
npm run build
```

ローカルプレビュー用ビルド

```bash
npm run preview
```
